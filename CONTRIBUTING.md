## Running Your Own Agentathon

### Prerequisites

Before running an Agentathon, ensure you have:

- [ ] Access to Microsoft Copilot Studio
- [ ] Azure subscription for additional resources (if needed)
- [ ] Identified target customer or internal team
- [ ] Secured venue (physical or virtual)
- [ ] Allocated facilitator(s) with Copilot Studio experience

### Step-by-Step Guide

#### 1. Planning Phase (4-6 weeks before)

| Week | Activity |
|------|----------|
| Week 6 | Identify customer/team and initial scope |
| Week 5 | Schedule kickoff call and define objectives |
| Week 4 | Send save-the-date and prerequisites checklist |
| Week 3 | Environment setup and license verification |
| Week 2 | Distribute pre-reading materials |
| Week 1 | Final logistics confirmation |

#### 2. Event Preparation

- [ ] Confirm participant list (ideal: 10-25 participants)
- [ ] Prepare Copilot Studio environments
- [ ] Set up Teams/communication channels
- [ ] Prepare presentation materials
- [ ] Create hands-on exercise scenarios
- [ ] Arrange catering (if in-person)

#### 3. During the Event

**Day 1: Discovery & Design**
```
09:00 - 09:30  Welcome & Introductions
09:30 - 10:30  Introduction to AI Agents & Copilot Studio
10:30 - 10:45  Break
10:45 - 12:00  Use Case Discovery Workshop
12:00 - 13:00  Lunch
13:00 - 14:30  Agent Design Session
14:30 - 14:45  Break
14:45 - 17:00  Hands-on Building (Part 1)
17:00 - 17:30  Day 1 Recap
```

**Day 2: Build & Demo**
```
09:00 - 09:30  Day 2 Kickoff
09:30 - 12:00  Hands-on Building (Part 2)
12:00 - 13:00  Lunch
13:00 - 15:00  Testing & Refinement
15:00 - 15:15  Break
15:15 - 16:30  Demo Presentations
16:30 - 17:00  Feedback & Next Steps
```

#### 4. Post-Event Activities

- [ ] Collect feedback from participants
- [ ] Document learnings and outcomes
- [ ] Create success story (if applicable)
- [ ] Define production roadmap with customer
- [ ] Share results with this repository

## Sharing Success Stories

### Success Story Template

When sharing a success story, please include:

```markdown
# [Customer Name] - Agentathon Success Story

## Overview
- **Customer**: [Name]
- **Industry**: [Industry]
- **Date**: [Event Date]
- **Location**: [City, Country]

## Challenge
[Describe the business challenge]

## Solution
[Describe the agents built during the Agentathon]

## Results
[Quantifiable outcomes if available]

## Testimonial
[Quote from customer, if available]

## Learnings
[Key insights from the event]
```

### Submitting Your Story

1. Create a new file in `success-stories/[country]/[customer-name].md`
2. Follow the template above
3. Include any screenshots or images in `assets/images/`
4. Submit a Pull Request
