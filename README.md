## ï¿½ğŸ“– What is an Agentathon?

An **Agentathon** is a **1-2 day hands-on workshop** where participants learn to design, build, and deploy AI agents using **Microsoft Copilot Studio**. Developed by the Microsoft COI (Copilot Onboarding Initiative) Team in Spain, this format has helped numerous organizations accelerate their AI adoption journey.

### Key Characteristics

- **Duration**: 1-2 days intensive workshop
- **Format**: Hands-on, collaborative building
- **Platform**: Microsoft Copilot Studio
- **Output**: Working AI agent prototypes
- **Audience**: Business users, IT professionals, developers

### What Participants Build

During an Agentathon, teams typically build agents for:
- ğŸ¦ **Customer Service**: Automated support and FAQ handling
- ğŸ‘¥ **HR Operations**: Employee onboarding, policy questions
- ğŸ”§ **IT Support**: Troubleshooting, ticket creation
- ğŸ“Š **Business Intelligence**: Data retrieval and reporting
- ğŸ“ **Process Automation**: Guided workflows and approvals
